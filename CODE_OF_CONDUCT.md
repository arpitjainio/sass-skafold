# Code of Conduct

## Our Standard

We want this project to be welcoming, respectful, and useful for people with different backgrounds and experience levels.

Examples of behavior that contribute to a positive environment include:

- being respectful in disagreements
- assuming good intent while still giving honest feedback
- welcoming questions from beginners and experienced contributors alike
- focusing critique on ideas, code, and decisions rather than people
- sharing context clearly and constructively

Examples of unacceptable behavior include:

- harassment, intimidation, or discrimination
- personal attacks, insults, or trolling
- deliberate intimidation or bad-faith disruption
- publishing private information without permission
- any conduct that would make participation feel unsafe

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or other contributions that violate this code of conduct.

## Scope

This code of conduct applies within project spaces, including issues, pull requests, discussions, and review comments.

## Reporting

If you experience or witness unacceptable behavior, open a private security-style report using the contact path described in [SECURITY.md](SECURITY.md), or contact the maintainers through a private channel you trust.
